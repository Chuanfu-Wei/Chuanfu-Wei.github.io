一、数列的极限

数列极限的性质

  • 唯一性：如果数列收敛，那么它的极限是唯一的
  • 有界性：如果数列收敛，那么数列一定有界

注意：逆命题数列有界，但数列并不一定收敛

  • 子数列收敛：如果数列收敛于a，那么他的任意子数列也收敛且极限也是a
  • 有序性：给定数列xₙ、yₙ，如果xₙ≤yₙ，且极限分别为a、b，那么a≤b
  • 保号性：数列存在极限a，且a>0，则存在正整数N，当n>N时，有Xₙ>0

推论：若Xₙ≥0(n=1,2,...)，且极限为a，则a≥0

数列极限的四则运算法则(n->∞)(有限项数列 )

  • lim(Aₙ+Bₙ) = limAₙ + limBₙ
  • lim(Aₙ·Bₙ) = limAₙ · limBₙ
  • lim(Aₙ/Bₙ) = limAₙ / limBₙ (limBₙ≠0)
  • 特殊：
    • lim(c·Aₙ) = c·limAₙ
    • lim(Aₙⁿ) = [limAₙ]ⁿ，ⁿ=1,2,...

数列收敛的判定方法

  • 夹逼定理：设xₙ≤aₙ≤yₙ(n=1,2,...)；且数列{xₙ}和数列{yₙ}收敛于相同的极限；则数列{aₙ}收敛，且limaₙ=limxₙ=limyₙ

常用结论：对于任何大于0的常数a，极限limⁿ√￣a=1

  • 单调有界原理：设数列{aₙ}单调增加(减少)且有上界(下界)，则数列{aₙ}存在极限；即 -> 任何单调有界数列一定存在极限
  • 区间套定理：设有区间序列{[aₙ,bₙ]}(ₙ=1)^∞满足；则存在唯一的x∈[aₙ,bₙ]（n=1,2...）
    1. [aₙ,bₙ] ⊇ [a(ₙ+1),b(ₙ+1)]，n=1,2...
    2. bₙ-aₙ -> 0，即bₙ,aₙ之间的长度趋近于零（n->∞）
